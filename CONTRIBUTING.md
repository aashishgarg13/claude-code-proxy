# Contributing to Claude Code Proxy

We love contributions! Here is how you can help:

## 🚀 Ways to Contribute
1.  **Add New Models**: Update the YAML files with new NVIDIA NIM or Gemini models.
2.  **Improve Scripts**: Make the bash scripts more robust (e.g., better error handling).
3.  **Add Skills**: Create new `.sh` files in the `skills/` directory.
4.  **Documentation**: Improve the README or add tutorials.

## 🛠️ Development Setup
1. Fork the repo.
2. Create a feature branch: `git checkout -b feat/my-new-feature`
3. Commit changes: `git commit -m "feat: add amazing feature"`
4. Push to the branch: `git push origin feat/my-new-feature`
5. Open a Pull Request.
